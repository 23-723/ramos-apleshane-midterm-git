# Midterm Examination Reflection

## What is Git used for?

Git is used to track the changes in the files and managing project history and it is a version control system. It also helps many developers to collaborate them without having a face-to-face meeting.

## What is the difference between Git and Github?

The difference between Git and Github is that git is a software that tracks the changes locally on the computer while the Github is an online platform that hosts Git repositories.

## Why is branching useful?

Branching is a useful to the developers to work on a new features without affecting the main branch or project. Once they add a new changes in new branch, they can be merged into the main branch safely.

## What command did you use to upload your local commits to Github?

The command that I use to upload into my local commits to Github is that `git push`. Because this command can synchronizes the local repository with the remote repository.
